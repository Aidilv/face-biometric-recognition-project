# Face Biometric Recognition Performance Analysis

## Project Title
Face Biometric Recognition Performance Analysis

---

## Research Problem
Face biometric recognition systems may produce incorrect recognition due to:

- Low similarity score
- Poor image quality
- Lighting condition
- Facial variation

These issues may reduce recognition accuracy and affect biometric system reliability.

---

## Research Objective
To analyse the performance of a face biometric recognition system using:

- Prediction result
- Similarity score
- Recognition accuracy

---

## Research Question
What is the accuracy level of the face biometric recognition system based on the tested dataset?

---

## Research Methodology

This project uses a quantitative experimental research approach.

The dataset was analysed using Python by comparing:

- Actual identity
- Predicted identity

The recognition result is classified as:

- Correct recognition
- Incorrect recognition

The system performance is evaluated using:

- Accuracy score
- Confusion matrix
- Classification report
- Similarity score analysis

---

## Dataset Description

The dataset contains the following columns:

| Column | Description |
|---|---|
| SampleID | Sample number |
| ActualIdentity | Real person identity |
| PredictedIdentity | Identity predicted by system |
| SimilarityScore | Recognition confidence score |
| RecognitionStatus | True or False recognition result |
